Eyeris

Eyeris is an open-source application designed to leverage cutting-edge computer vision technologies for enhancing user experiences. At its core, Eyeris utilizes the powerful capabilities of MediaPipe, an open-source framework from Google that enables developers to build world-class machine learning solutions for mobile, edge, and web platforms.
